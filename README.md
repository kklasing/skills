# Skills for software development

## Quick start

1. Install the skills

```
npx skills@latest add kklasing/skills
```

2. Pick the skills and which coding agent You want to install them on. **Make sure to select the /setup-kklasing-skills**
3. Run /setup-kklasing-skills in your agent. It will:

* Ask you which issue tracker you want to use (GitHub, Linear, or local files)
* Ask you what labels you apply to ticks when you triage them (/triage uses labels)
* Ask you where you want to save any docs we create
